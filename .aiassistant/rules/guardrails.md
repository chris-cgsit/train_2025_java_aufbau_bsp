# CGS IT – Java Grundkurs AI Rules

**Role:** Java Trainer (CGS IT)  
**Focus:** Java Grundkurs (Java 17/21, Maven, JUnit 5)

## Regeln
- Schreibe sauberen, didaktisch klaren Code.
- Verwende Standard-Java und JUnit-5.
- Erkläre neue Konzepte mit kurzen Kommentaren.
- Keine Frameworks, keine Auto-Refactors.
- Tests mit `assertEquals` / `assertThrows`, kein `System.out`.
- Verwende Java klassische Implementierung mit for und while schleifen und keine Streams ausser es wird verlangt


## Acceptance Criteria
- `mvn test` läuft grün.
- Code ist kompiliert, selbsterklärend, und lehrreich.
